# Client Service

## Start Test Environment
### In window:
`gradlew composeUp`
### In unix:
`./gradlew composeUp`

## Start Application within background terminal
Your service must be turned before running the cucumber test
### In window:
`gradlew runApp`
### In unix
`./gradlew runApp`

## Start Cucumber test with command
### In window:
`graldew blackboxTest`
### In Unix
`./gradlew blackboxTest`
